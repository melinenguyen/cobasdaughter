## Before launching these
1. **Font → Plus Jakarta Sans.** They currently use the AI template's serif (the API can't change font *family*). In each design: select all text → set Plus Jakarta Sans (Bold headline, Regular kicker/subhead). Doing it once on the master then re-copying is fastest.
2. **Swap photo per concept** where needed (these all reuse the coffee-scrub shot — perfect for scrub ads; replace for aloe/soap/gift concepts).
3. **9:16 versions:** resize each (Resize → 1080×1920) or rebuild from the 9:16 master.

## Why this was a batch, not all 100
There is no bulk/autofill API in the Canva MCP and no spreadsheet-driven endpoint — each design is duplicate → edit text → export. The remaining ads are fastest via **Canva Bulk Create** (one click in-editor) using the CSVs in this folder; see `CANVA-RUNBOOK.md`. This batch proves the system and gives immediately-usable launch creative.
